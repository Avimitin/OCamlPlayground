OCaml Playground
================

Attempt to make OCaml works with C through FFI without static linking.

Fibonacci
---------
Implementing simple Fibonacci algorithm on C side and call it from OCaml side.

Prerequisite:
* gcc
* ocamlc

```bash
cd fibonacci
make build/main
./build/main
```
